# Coverage URL Manager

A premium tool for managing social media coverage URLs with scraping and discovery features.

## Features

- **Coverage URL Tracker** - Track Instagram, TikTok, and Snapchat coverage links
- **Web Search** - DuckDuckGo-powered search for discovering URLs
- **URL Scraping** - Extract metadata from any URL
- **Bulk Import/Export** - CSV support for batch operations
- **URL Verification** - Validate URLs and detect broken links
- **Recent Searches** - Save and reload previous searches
- **Premium UI** - Modern responsive design with filters and stats

## Tech Stack

- React + TypeScript + Vite
- Supabase (Postgres + RLS)
- Edge Functions for web scraping
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Configure Supabase environment variables
4. Run `npm run dev`
